# Редизайн Lamoda для групп web-разработки :rocket:

## Требования

> PHP 7.4+
>
> node 16+
>
> MySQL 8.0+

## Установка

- Клонируем проект

```
git clone https://github.com/ToltekPlus/lamoda.git .
```

- Разворачиваем Docker
```
docker-compose up -d
```

- Запускаем в браузере
```
http://localhost:8000
```
